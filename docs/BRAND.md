# QATQ Brand Notes

The recommended logo direction is a technical mark, not a mascot:

- a `Q` ring for the project name;
- four small nodes for quaternion structure;
- compressed vertical byte-plane bars for the exact codec surface;
- dark neutral background with teal and amber accents.

The starter SVG is `assets/qatq-logo.svg`. It is intentionally simple so it can
be rendered in README files, release notes, and social cards without requiring a
design system.

Recommended GitHub/social description:

> Quaternion-grade compression for LLM KV caches and live runtime migration.

Recommended topics:

- `rust`
- `compression`
- `llm`
- `kv-cache`
- `tensor`
- `quantization`
- `zstd`
- `llama-cpp`
- `migration`
- `machine-learning`
