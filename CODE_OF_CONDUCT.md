# Code of Conduct

QATQ follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Standards

Examples of behavior that contributes to a positive environment:

- using clear, technical, evidence-based feedback;
- respecting different experience levels and areas of expertise;
- accepting responsibility for mistakes and correcting them;
- focusing criticism on code, claims, and process rather than people.

Unacceptable behavior includes harassment, discriminatory language, personal
attacks, sexualized attention, doxxing, or sustained disruption of project
work.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and
pull requests that are not aligned with this Code of Conduct. Maintainers may
also temporarily or permanently ban contributors for repeated or severe
violations.

Report enforcement concerns to the repository owner. Reports will be handled as
privately as practical.
