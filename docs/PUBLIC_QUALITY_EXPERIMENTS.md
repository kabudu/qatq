# QATQ Quality Experiments

Generated by `qatq-bench --quality-output`. These paper-facing codec probes compare the base `turboquant-q4` QJL inner-product estimator with the quaternion-overlay `phase1-q4` decoded inner product on deterministic finite query probes.

This is not a model-quality or perplexity result. It is a deterministic codec-level proxy for query/dot-product preservation over the selected fixture corpus.

## Method

- Each dataset is evaluated with four deterministic query probes: normalized self, alternating signs, sinusoidal, and sparse signs.
- Non-finite source values are masked to zero for this quality-only probe so NaN/Inf exactness stress fixtures do not poison finite inner-product metrics.
- `turboquant-q4` is scored through `estimate_turboquant_q4_inner_product`, which uses the stored QJL residual signs.
- `phase1-q4` is scored by decoding the quaternion-overlay payload and taking the decoded inner product.
